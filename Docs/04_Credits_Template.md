# Credits Template

이 파일은 외부 에셋, 폰트, 사운드, 라이브러리의 출처를 기록하기 위한 템플릿이다.  
실제 개발 중에는 이 파일을 복사해 `Credits.md`로 사용한다.

---

## 작성 규칙

- 외부 자료를 프로젝트에 넣는 즉시 기록한다.
- "무료"라고만 쓰인 자료는 사용하지 않는다. 반드시 라이선스 이름을 확인한다.
- CC-BY 자료는 저작자 표시가 필수다.
- CC0 자료도 추적을 위해 기록한다.
- 라이선스가 불명확하면 사용하지 않는다.

## License Quick Guide

| License | Use? | Rule |
|---|---|---|
| CC0 | Yes | 상업 사용/수정 가능. 기록 권장. |
| CC-BY | Yes | 상업 사용 가능. 크레딧 필수. |
| MIT | Yes | 코드/라이브러리에 적합. 라이선스 고지 권장. |
| OFL | Yes | 폰트에 적합. 라이선스 파일 유지 권장. |
| Unity Asset Store Standard EULA | Usually Yes | 게임에 포함 가능. 원본 에셋 단독 재배포 금지. |
| CC-BY-NC | No | 상업 사용 금지. |
| CC-BY-ND | No | 수정 금지. 게임 에셋에 부적합. |
| GPL | Avoid | 프로젝트 공개 의무 등 관리가 어렵다. |
| Unknown | No | 라이선스 불명확. |
